## Packages
framer-motion | Smooth animations for page transitions and layout changes
date-fns | Formatting dates for tasks and boards

## Notes
Authentication is handled by Replit Auth via /api/login and /api/logout
The app uses a split-screen landing page design
Board view implements a 3-column Kanban layout
Status updates trigger layout animations
